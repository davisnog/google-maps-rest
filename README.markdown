Pet Project - Google Maps REST Api
====================================

Projetinho que uso para fazer o parse do json recebido do Google Maps API usando como base 
a url: http://maps.googleapis.com/maps/api/geocode/json?address=endereço que eu quero pesquisar&sensor=false

Estou usando basicamente para pegar a latitude e longitude de um determinado endereço.
